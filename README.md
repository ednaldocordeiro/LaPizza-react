  ## 🖇 Executar o Projeto
  
- Clone o repositório em sua máquina:
    ```bash
    $ git clone https://github.com/Ednaldo-byte/LaPizza-react.git 
    ```
- Entre no repositório usando:
    ```bash
    $ cd LaPizza-react
    ```
- Execute o comando para instalar as dependências:
    ```bash
    $ yarn
    ```
- Para iniciar a aplicação, use:
    ```bash
    $ yarn dev
    ```
